Práctica de GIT

comando Descripcion
git init Inicializa la carpeta como un repositorio de GIT, para empezar a gestionar el cambio
git log Permite visualizar todos los commits realizados en un repositorio
git add -A Agrega todos los archivos que han sido modificados o archivos nuevos del working área al stagging
git commit -m "" pasa los archivos del área de stagging al área del repositorio, es necesario poner un mensaje que represente todos los cambios del proyecto
git push origin master manda los archivos del área de repositorio a un servidor, por ejemplo github, bitbucket, gitlab, etc.
git checkout este comando da saltos de gits
